# temp_linmot_ec

This repo will be transferred over to the mono-repo once it has been confirmed as finished and able to reach version 1.0.

Our company, Smart Vision Works, is attempting to write custom software for the LinMot device LinMot C1250-EC. LinMot support has given us their official Python codebase used to control the motor via EtherCAT.

We must port the Python library into GoLang modules for use in our stack.

Read the Markdown docs in `stage_linmot_app`, `stage_linmot_ct`, `stage_linmot_drive`, and `stage_linmot_proto` to understand what we're trying to do here.
